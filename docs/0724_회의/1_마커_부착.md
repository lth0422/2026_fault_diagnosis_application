# 1. 마커 부착 관련

## 결론
마커 부착은 현재 시스템의 **필수 전제이자 현장 적용의 가장 큰 제약**이다. 최근 연구는 **마커리스(markerless)**로 수렴 중이며, 물리적 타당성도 입증되어 있어 우리도 중장기적으로 마커 제거를 검토할 가치가 있다.

## 현재 방식
- 형광/대비색 마커를 회전기계 표면에 부착 → OpenCV HSV 범위로 마커 분리 → 마커 중심 변위(DisplacementZ) 추적.

## 한계
| 항목 | 내용 |
|------|------|
| 물리적 이탈 | 고속 회전체에서 마커가 떨어질 수 있음 |
| 색상 변질 | 열·유체·오염 환경에서 HSV 색상이 변해 검출 실패 |
| 부착 불가 | 표면이 작거나 접근 불가한 부위엔 부착 자체가 어려움 |
| **모델 오작동** | 마커 검출 실패 시 DisplacementZ ≈ 0 → **모든 결과가 H(정상)로 쏠림** (개발 중 실제 확인) |

## 문헌 근거 (마커리스는 실현 가능)
- **Phase-based video motion processing** [1]: 마커리스 위상기반 모션처리의 원조. optical flow 계산 없이 미세운동 추출. **망치 가진 실험에서 영상 위상→변위→가속도가 접촉식 가속도계와 일치**(원문 Fig.9) → 카메라 변위의 물리적 타당성 입증.
- **영상 위상차 기반 진단** [2]: 우리와 거의 동일 구조인데 **마커 대신 베어링 seat 엣지의 Gabor 위상차**를 사용해 4클래스 진단.
- **서브픽셀 visual vibrometry** [3]: 마커 없이 저프레임레이트로 **1/200픽셀(0.65μm)** 분해능 달성.
- 종합하면, 마커기반 → **마커리스(feature tracking / optical flow / DIC)**로 무게중심이 이동 중 [1][2][3].

## 마커리스 대안 후보
| 방법 | 특징 | 난이도 |
|------|------|--------|
| Phase-based motion [1] | 미세진동, 노이즈 강건 | 높음(온디바이스 계산부하) |
| Optical flow (Lucas-Kanade) | 특징점 추적 | 중간 |
| 템플릿 매칭 | 국소 패턴 추적 | 중간 |
| Keypoint(ORB/SIFT) | 자연 특징점 | 중간 |

## 회의 논의점
- [ ] 1차 논문은 **마커 유지**, 마커리스는 후속/옵션으로 둘 것인가?
- [ ] 마커리스를 "한 방"으로 밀 경우 — **고속 회전체에서 특징점 유지**가 핵심 난제 [2].
- [ ] 마커 검출 실패 → H 쏠림 문제는 **신뢰도 표시/방어 로직**으로 우선 완화 가능(앱 확장).
