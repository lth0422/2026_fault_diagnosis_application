# 2. 정확도가 비정상적으로 높은 것 관련

## 결론
**99.7% 정확도는 데이터 누수(data leakage)로 부풀려졌을 가능성이 매우 높다.** "높아서 오히려 의심스럽다"는 직관이 문헌으로 뒷받침된다. **정직한 재평가**가 저널화의 최우선 과제이며, 이 재평가 자체가 논문 기여가 될 수 있다.

## 두 가지 누수가 동시에 존재
### ① Segmentation-level 누수 (확정적)
- 같은 영상에서 슬라이딩 윈도우(창 2048, **중첩 1024**)로 자른 샘플을 10-fold에 **무작위**로 나눔.
- 중첩 1024면 인접 샘플이 50%를 공유 → 거의 같은 데이터가 train/test 양쪽에 [2].

### ② Bearing-level 누수 (더 근본적)
- 우리는 **클래스당 물리 베어링이 1개**(IR 1, OR 1, B 1, 정상 1).
- "그 베어링 고유 signature"와 "클래스 라벨"이 완벽히 상관 → 모델이 **결함 유형을 배운 건지 그 베어링을 외운 건지 원리적으로 구분 불가** [1][2].
- 베어링이 1개뿐이라 **bearing-wise 분할이 아예 불가능** → 일반화 능력을 현재 데이터로는 증명 못 함 [2].

## 문헌 증거
| 근거 | 내용 |
|------|------|
| **데이터 누수 영향** [1] | 베어링 단위(part-to-part) 분할로 바꾸면 **정확도 최대 47%p 하락**. 선행 55편 중 다수가 이 문제 |
| **평가 프로토콜** [2] | 누수 3유형(segmentation / bearing-condition / bearing-repetition) 정의, **bearing-wise 분할 + Macro AUROC** 권장. 학습 베어링 다양성이 일반화 결정 요인 |
| **WDCNN 원저자 관행** [3][4] | Zhang(우리 모델 원조)은 **overlap 증강을 train에만, test는 비중첩**으로 유지 — WDCNN [3]·TICNN [4] 양쪽 명시. 우리가 전체 증강 후 무작위 분할했다면 원저자보다 느슨 |
| **가속도계 비교 방증** [5] | 엄격한 연구: 영상-비전 **95~98%** < 접촉식 가속도계 **98.9~99.5%**가 정상. 우리 영상 기반 99.7%는 가속도계급 초과 → 물리적으로 이상 |
| **최유사 논문도 동일 함정** [6] | Peng(99.792%)도 단일베어링+슬라이딩윈도우+무작위분할 → 같은 위험 |

## 재평가 계획 (현재 데이터로 가능)
| 조건 | 분할 방법 | 예상 |
|------|-----------|------|
| S0 | 전체 중첩증강 후 random 10-fold (현행 재현) | ~99.7% |
| S1 | 중첩 제거 + 시간순 blocked | 하락 |
| S2 | recording-wise (영상 단위 분리) | 더 하락 |
| S3 | (가능시) bearing-wise | 최저 = 진짜 일반화 |

- 지표도 accuracy → **balanced accuracy / Macro AUROC**(prevalence-independent)로 교체 [2][8].
- multi-label 이진 공식화(IR/OR/B 독립 detector, 정상=전부0) 검토 [2][8].
- 동일 베어링이 train/test에 걸치면 과도낙관이라는 지적은 다수 문헌에서 반복 확인됨 [7][8].
- **S0→S3 곡선 자체가 논문의 정직한 기여.**

## 근본 해결
- 재분할로는 완치 안 됨(베어링 1개). **클래스당 복수 물리 베어링 추가 촬영**이 필요(최소 3개, bearing-wise 가능화) [1][2].
