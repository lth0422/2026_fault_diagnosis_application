# 4. 저널 논문 상향을 위한 노블티(Novelty)

## 결론
**"스마트폰 + 베어링 결함 분류 + 온디바이스 딥러닝"의 교집합은 문헌상 비어 있다.** 이것이 우리 자리다. 노블티의 핵심 축은 **접근성(별도 장비 불요)**이며, 이는 이미 top 저널에서 인정된 가치이자 future work로 지목된 방향이다.

## 노블티 포지셔닝
| 축 | 선행 연구 대다수 | 우리 |
|----|------------------|------|
| 카메라 | 산업 고속카메라(1000~5000fps) / 이벤트 카메라 (고가·특수) | **소비자 스마트폰 240fps** |
| 대상 | 스마트폰은 구조물(SHM)·스포츠·회전속도에 사용 | **베어링 4클래스 결함 분류** |
| 배포 | 대부분 오프라인 후처리 | **온디바이스 앱(Android/iOS) End-to-End** |
| 모델 | 최근 7B LLM 등 무거움 | **경량 WDCNN(1.5M급) 온디바이스** |

- **핵심 gap:** 이 세 조건의 교집합을 다룬 논문이 사실상 없음.
  - SURVISHNO(MSSP 2021): 스마트폰 진동 분석 → 하지만 **속도추정·모달, 결함분류 아님**.
  - Peng(MSSP 2023)/Zhou(MST 2026): 영상 베어링 진단 → 하지만 **산업 고속카메라**.
  - 이벤트카메라(IEEE TII 2024)/멀티모달LLM(Sensors 2025): 베어링 진단 → 하지만 **특수 하드웨어+무거운 모델**.

## 노블티를 뒷받침하는 근거
| 근거 | 내용 |
|------|------|
| **접근성은 인정된 가치** | SURVISHNO(MSSP 2021): "주머니 속 스마트폰 = 저비용 진동 분석 장비" — top 저널에서 공인 |
| **우리 방향을 저널이 지목** | Yu & Lubineau(Measurement 2021) 결론: *"스마트폰 960fps 고속촬영 + 앱 기반 실시간 추적을 향후 탐색할 가치가 있다"* → 우리가 바로 그것 |
| **경제성** | 저비용 진단이 상용 대비 $500 vs $8,000~45,000(Vanraj 2018). 우리는 추가 하드웨어 **$0** |
| **배포성** | 이벤트카메라·7B LLM 대비 스마트폰+경량 WDCNN이 현장 배포에 압도적 유리 |

## 저널 "한 방" 후보 (Options)
| 옵션 | 내용 | 특징 |
|------|------|------|
| A. 마커리스 | phase/optical flow로 마커 제거 | 임팩트 큼, 난이도 높음 |
| B. 다조건 강건성 데이터셋 | RPM·조명·기기·각도 체계적 실험 | 실현 가능성 높음, 취약점 정면 대응 |
| C. 도메인 적응 | 조건 달라도 성능 유지 | 실용성 강화 |
| D. XAI 해석 | Grad-CAM으로 결함주파수 기여 시각화 | 빠름, 신뢰성↑ |
| E. 정직한 평가 프로토콜 | 누수 없는 bearing-wise 평가 도입 | 방법론 기여, 우리 강점과 직결 |

## 논문 기여 3점 (제안)
1. **접근성 있는 스마트폰 온디바이스 베어링 진단 시스템**(앱) — 별도 장비 불요.
2. **다조건 강건성 + 측정 불확도 정량 보고** — 실용성·신뢰성.
3. **카메라 기반 진단에 누수 없는 평가 프로토콜 도입** — 방법론.
