# fps · 시간해상도 분석 (스마트폰 vs 기존 카메라 기반)

핵심 질문: **"우리는 스마트폰 240fps라 기존 카메라 기반과 근본적으로 다르다. fps가 부족하다."**
정독한 논문들의 촬영 fps와 베어링 결함주파수를 비교해, 이 차이가 **얼마나 근본적인지** 정량화한다.

> 결론 먼저: **240fps(Nyquist 120Hz)로는 대부분의 베어링 결함 특성주파수(BPFO/BPFI, 보통 100~200Hz+)와
> 그 고조파·충격 공진(수 kHz)을 원리적으로 포착할 수 없다.** 기존 영상 진단은 1000~5000fps 또는
> 이벤트 카메라를 쓴다. 이는 우리의 근본적 한계이자 동시에 차별점(저비용·저fps 영역의 개척)이다.

스마트폰(30~240fps)은 **회전속도·구조 변위·모달**에 쓰였지 베어링 결함 분류엔 안 쓰였다.

## 2. 왜 fps가 결정적인가 — 베어링 결함주파수

베어링 결함은 굴림체가 결함부를 지날 때 발생하는 **특성주파수(characteristic frequency)**로 나타난다:

- BPFO(외륜) = (n/2)(1 − (d/D)cosθ)·f_r
- BPFI(내륜) = (n/2)(1 + (d/D)cosθ)·f_r
- BSF(볼)   = (D/d)(1 − ((d/D)cosθ)²)·f_r

(n=굴림체 수, d=굴림체경, D=피치경, θ=접촉각, f_r=회전주파수)

### B3(Zhou) 베어링 기준 계산 (n=11, d=7.94, D=33.5, θ=0)

| RPM | f_r | BPFO | BPFI |
|-----|----:|-----:|-----:|
| 1800 | 30 Hz | ~126 Hz | ~204 Hz |
| 1200 | 20 Hz | ~84 Hz | ~136 Hz |

- B3가 **5000fps(Nyquist 2500Hz)**를 쓴 이유: BPFO/BPFI(100~200Hz)뿐 아니라 **고조파와 충격 공진**까지 잡으려면 필요.
- **우리 240fps(Nyquist 120Hz)**: 1200rpm에서 BPFO(~84Hz)는 겨우 잡히나 **BPFI(~136Hz)는 Nyquist 초과 → 앨리어싱/손실**. 고조파는 전부 손실.
- **속도를 올리면(≥1500rpm) BPFO조차 120Hz를 넘어 못 잡는다.** → 우리가 1200rpm 단일조건을 쓴 것이 사실상 **fps 천장에 걸린 결과**일 수 있다.

> 즉 M3가 1000fps를 쓰고도 **1×/2×(24~48Hz)만** 분석한 것처럼, 영상 기반은 태생적으로
> **저주파 구조/회전 진동**에 강하고 **고주파 베어링 임펄스**에 약하다. 스마트폰은 이 제약이 훨씬 심하다.

## 3. 그럼 우리는 무엇을 측정하고 있나?

이 분석은 우리 방법의 물리적 정체성을 다시 보게 한다.

- 우리 입력은 **DisplacementZ(마커 중심 변위)**, 가속도가 아니다.
- 변위 = 가속도/ω² → **변위 신호는 본질적으로 저주파를 강조**(고주파는 1/ω²로 감쇠).
- 즉 **저fps(저Nyquist) + 변위기반**은 둘 다 저주파를 본다는 점에서 **서로 일관**된다.
- 따라서 우리가 실제로 포착하는 것은 **회전 기본주파수와 저차 성분**(불평형·정렬·저차 BPFO)일 가능성이 크고,
  고주파 베어링 임펄스가 아니다.

**핵심 과학적 질문(논문 방어 지점):**
1. 저주파 변위 대역에 **베어링 유형(IR/OR/B)을 구분할 판별 정보가 실제로 있는가?**
2. 있다면 어떤 물리량(저차 BPFO, 결함에 의한 회전 변조)인가?
3. 없다면 99.7%는 결함 물리가 아니라 **누수/아티팩트**를 학습한 것 → [data_leakage_analysis.md] 와 직결.

## 4. 논문에서의 프레이밍 (한계이자 차별점)

**정직한 한계로:**
- "스마트폰 240fps는 고주파 베어링 특성주파수를 포착하지 못한다. 따라서 본 방법의 적용 범위는
  저주파 변위가 결함을 반영하는 저·중속(≤~1200rpm) 조건으로 제한된다."

**차별점/기여로:**
- 기존 영상 진단은 고가 고속·이벤트 카메라(1000~5000fps)에 의존 → 현장성·접근성 낮음.
- 우리는 **의도적으로 저fps·저비용 영역**에서 "그래도 진단이 되는가?"를 규명 → 미개척 영역.
- 저주파 변위만으로 4클래스가 실제로 구분되는지 **정직하게(누수 없이) 검증**하는 것 자체가 기여.

**실험 설계 함의:**
- RPM 강건성 실험 시 **240fps Nyquist(120Hz) 대비 BPFO/BPFI 위치를 RPM별로 표기** → 어느 속도까지
  물리적으로 유효한지 경계를 명확히 제시.
- 가능하면 **960fps(최신 스마트폰) 모드**와 비교(Yu 2021이 future work로 지목) → fps↑시 성능 변화 정량화.
